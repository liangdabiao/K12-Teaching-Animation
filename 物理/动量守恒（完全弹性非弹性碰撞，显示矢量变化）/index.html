动量守恒定律：碰撞可视化教学动画

动量守恒定律：系统不受外力或合外力为零时，总动量保持不变。公式：m₁v₁+m₂v₂ = m₁v₁'+m₂v₂'。完全弹性碰撞：动量守恒+动能守恒；完全非弹性碰撞：碰后共速，动能损失最大。

核心公式:
动量守恒: m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
完全弹性碰撞动能守恒: (1/2)m₁v₁² + (1/2)m₂v₂² = (1/2)m₁v₁'² + (1/2)m₂v₂'²
完全非弹性碰撞后共同速度: v' = (m₁v₁ + m₂v₂) / (m₁ + m₂)
