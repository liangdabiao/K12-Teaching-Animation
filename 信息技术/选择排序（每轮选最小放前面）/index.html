选择排序算法动画演示

每轮选择最小值放到前面 - 可视化理解排序过程

算法原理

选择排序是一种简单直观的排序算法。它的工作原理是：

1. 首先在未排序序列中找到最小（大）元素，存放到排序序列的起始位置。

2. 再从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾。

3. 重复第二步，直到所有元素均排序完毕。

本动画演示的是每轮选择最小值放到前面的版本。

伪代码

for i = 0 to n-1:
  minIndex = i
  for j = i+1 to n:
    if array[j] < array[minIndex]:
      minIndex = j
  swap(array[i], array[minIndex])
