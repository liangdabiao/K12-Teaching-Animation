原理说明

物质的量(n)是国际单位制7个基本物理量之一，单位是摩尔(mol)。1mol含有阿伏加德罗常数Nₐ≈6.02×10²³个微粒。摩尔质量(M)：1mol物质的质量(g/mol)，数值=相对分子质量。标准状况下(S.T.P: 0°C,101kPa)气体摩尔体积Vm=22.4L/mol。

● 分子/原子● 1mol集合体● 气体22.4L

核心公式

  • n = N / Nₐ （Nₐ = 6.02×10²³ mol⁻¹）
  • n = m / M （m:质量g，M:摩尔质量g/mol）
  • n = V / Vm （S.T.P下Vm = 22.4 L/mol）
  • c = n / V （c:物质的量浓度 mol/L）
  • 例：1mol H₂O = 18g = 6.02×10²³个分子
  • 例：S.T.P下 1mol O₂ = 22.4L = 32g
